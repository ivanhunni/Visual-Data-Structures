# Visual Data Structures

An interactive web application that visually demonstrates how common **data structures** work.
The goal of this project is to help users understand operations step-by-step through **animations and visual explanations**.

## Features

* Interactive visualization of data structures
* Step-by-step animation of operations
* Control animation speed
* View pseudocode for operations
* See time complexity (Big O)
* Perform operations manually

## Supported Data Structures

Currently implemented:

* Stack
* Queue
* Array

Planned:

* Linked List
* Binary Tree
* Heap
* Graph

## Operations

### Stack

* Push
* Pop
* Peek

### Queue

* Enqueue
* Dequeue

### Array

* Insert
* Delete
* Access by index

## How It Works

Each operation is broken into **multiple visual steps**.
Instead of instantly changing the structure, the app generates a sequence of states that are animated to show exactly what happens internally.

Example:

Push to stack:

1. Highlight the value being inserted
2. Show the new element appearing
3. Move it to the top of the stack
4. Update the stack state

This makes the behavior of data structures easier to understand.

## Animation Controls

Users can control the visualization with:

* Step forward
* Step backward
* Play animation
* Pause animation
* Adjust animation delay
